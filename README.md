## Haber Sitesi Projesi
Bu projede .net core ortamında Haber Sitesi yapılmıştır.
### Kullanılan Araçlar ve Teknolojiler<hr>
 1. Çok Katmanlı Mimari
 2. Nesne Yönelimli Programlama
 3. Asp.Net Core MVC
 4. Entity Framework Core (Code First)
 5. Ninject
 6. Fluent Validator
 7. CKEditör
